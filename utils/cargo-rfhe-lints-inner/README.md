# RFHE-lints-inner

Implementation of the lints in the custom [rfhe-lints](../cargo-rfhe-lints/README.md) tool.

## Installation
```
cargo install --path .
```

## Usage
This can be run directly by specifying the toolchain:
```
cargo +nightly-2024-05-02 rfhe-lints-inner
```

For a more ergonomic usage, see [rfhe-lints](../cargo-rfhe-lints/README.md)
